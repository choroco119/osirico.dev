# RoomLayoutDesigner 複製機能の挙動改善

Ctrl+ドラッグによるオブジェクト複製操作において、より直感的でスムーズな操作感を実現するための修正を行いました。

## 変更内容

### 1. 複製後のドラッグ対象切り替え
Ctrl+ドラッグで複製が行われた際、マウスで操作している対象を「元のオブジェクト」から「新しく作成されたオブジェクト」へと自動的に切り替えるようにしました。
これにより、ユーザーはマウスを止めることなく、そのままコピーした家具を引き出して配置することができます。

### 2. 移動中の一時的な衝突判定無効化
複製された直後のドラッグ操作に限り、そのオブジェクトの衝突判定を一時的に無効化（`tempDisableCollision`）するようにしました。
- 元の家具と重なっている状態からでも、引っかかることなくスムーズに引き出すことができます。
- マウスを離す（ドラッグを終了する）とフラグが解除され、通常の衝突判定（赤色表示など）が有効に戻ります。

### 3. 命名規則の維持
既存の自動名称（数値加算）のロジックを維持しつつ、複数選択時の複製にも対応しています。

## 検証結果

ブラウザ subagent を使用して以下の動作を確認しました：
- [x] オブジェクトを Ctrl+ドラッグ した際、新しいコピーがマウスに追従すること
- [x] ドラッグ中、他の家具と重なっても赤くならず（衝突判定がOFF）、自由に移動できること
- [x] マウスを離すと衝突判定が復帰し、再度重ねると衝突を検知すること
