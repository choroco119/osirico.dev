# 背景画像のサイズ・透過度調整機能 完了報告

読み込んだ背景画像の視認性や大きさをユーザー自身で微調整できる機能の実装が完了しました。

## 実装内容まとめ

1. **背景画像 調整パネルの追加**
   - 背景画像を読み込むと、画面左側の「パーツ（家具を追加などが並んでいるエリア）」の一番下に「背景画像 調整」というパネルが動的に表示されます。
   - 画像が読み込まれていない場合は、設定項目自体が隠れるようにスッキリとしたUI設計にしました。

2. **背景サイズ（縮尺）の変更**
   - 調整パネル内の「サイズ (%)」数値を変更することで、元画像を基準 (100%) とした縮尺調整が可能です。
   - `%` 単位での入力によって、背景画像の絶対的なサイズ基準を細かく合わせられます。
   - このサイズ指定は全体表示のズームやパンに連動して保持されます。

3. **透過度 (Opacity) の変更**
   - 「透過度 (0-100%)」のスライダーを左右に動かすことで、背景の濃さを自在に変更できます。
   - 「壁を書く時は少し濃くして見えやすくする」「家具を配置する時は薄くして、間取りの下書き感を出したい」といった使用目的に合わせた切り替えが可能です。

4. **JSON保存への対応**
   - これらの「背景画像のサイズ」「透過度」の情報も `STATE.config` に保持されるため、これまで通り `保存 (JSON)` に設定データが引き継がれ、リロードや `読込 (JSON)` 時に適用内容が復元されます。
