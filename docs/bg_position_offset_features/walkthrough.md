# 背景画像位置調整機能 完了報告

読み込んだ背景画像の配置位置（X / Y オフセット）を直接調整する機能を追加しました。

## 実装内容まとめ

1. **背景画像 調整パネルの拡張**
   - 画面左側の「背景画像 調整」パネルの中に、「X位置」および「Y位置」の数値入力欄を追加しました。

2. **位置オフセットの適用**
   - X位置・Y位置ともにミリメートル基準で平行移動（Translate）できるため、家具や壁の基準座標系に正確に合わせることができます。
   - スケール(縮尺)変更を同時に入れても、位置ベクトルに自動的に連動するため、画像の基準位置を合わせつつ拡大・縮小するといった操作が可能です。

3. **保存への対応**
   - 追加したX位置とY位置の情報は `STATE.config` に引き継がれ、`保存 (JSON)` に出力され、次回起動時の `読込 (JSON)` にて正確な箇所に復元されます。
