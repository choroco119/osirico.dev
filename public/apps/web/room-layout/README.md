# ルームレイアウトデザイナー 使い方

本ツールは、ブラウザ上でお部屋のレイアウトをシミュレーションし、家具や設備の配置を検討するためのツールです。

## 主な機能

- **自由な間取り作成**: クリックとドラッグで、好きな形状の部屋（壁）を作成できます。
- **オブジェクト配置**: 家具や設備、窓・ドアなどのパーツを実寸サイズで配置可能。
- **衝突判定**: オブジェクト同士や壁との重なりを自動で検知し、赤色で警告を表示。
- **マルチデバイス対応**: PCでのマウス操作はもちろん、スマホやタブレットでのタッチ操作にも対応。

## 基本的な操作方法

### 部屋の作成（壁）
- **頂点の移動**: 赤い丸（頂点）をドラッグして、壁の形状を自由に変えられます。
- **頂点の追加**: 壁の辺（白い線）をタップ/クリックすると、その位置に新しい頂点が追加されます。
- **頂点の削除**: 頂点を選択し、プロパティパネルまたはキーボードの `Delete` で削除できます（最低3つの頂点が必要です）。

### オブジェクト操作
- **追加**: 左メニューの「パーツ」からオブジェクトを選択してキャンバスに追加。
- **移動**: オブジェクトを直接ドラッグ。
- **複製**: `Ctrl + ドラッグ`（PC）またはプロパティパネルの「複製」ボタン。
- **多重選択**: `Shift + クリック`（PC）で複数のオブジェクトをまとめて操作。

### スマートフォン・タブレット（タッチ操作）
- **タブ切り替え**: 上部のナビゲーションバーで「パーツ」「キャンバス」「プロパティ」「操作」を切り替え。
- **パン/ズーム**: 背景を2本指でスワイプして移動、ピンチ操作で拡大縮小。
- **操作ガイド**: 「操作」タブから、いつでも簡易的な操作案内を確認できます。

## ヒントと注意点
- **初期化**: 画面右上の「クリア」ボタンですべてのオブジェクトを削除できます（壁は残ります）。
- **座標固定**: プロパティパネルの「位置を固定」にチェックを入れると、パーツが動かなくなります。
